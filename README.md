 Guia rápido: Simular o ESP32 no Wokwi

Use este link para abrir a simulação: https://wokwi.com/projects/447541018548864001

## Passos fáceis para usar ##

Clique ou copie/cole o link acima no navegador.
Ele abrirá a simulação pronta no Wokwi.

Quando a página abrir, aguarde a simulação carregar. Você verá um esquema do ESP32 e componentes.

No editor que aparece, você pode ver o código (geralmente sketch.ino) e o diagrama dos fios.

Para rodar a simulação, clique no botão "Play" ou ► Start Simulation (ou similar) no topo da página.

Observe as logs na saída, e a animação movimentando o servo motor.
